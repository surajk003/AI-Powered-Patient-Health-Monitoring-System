# 🩺 AI Powered Patient Health Monitoring System

An intelligent healthcare monitoring application that predicts potential health conditions using **Machine Learning** and provides **AI-generated health insights using Google Gemini AI**.

The system analyzes patient vital signs such as heart rate, SpO2 level, blood pressure, body temperature, and fall detection to predict possible health conditions.

This project combines **Machine Learning, Artificial Intelligence, and GUI-based healthcare monitoring** to assist in early health analysis.

---

# 🚀 Features

✔ Patient vital monitoring
✔ Machine learning disease prediction
✔ Random Forest trained model
✔ AI health insights using Google Gemini
✔ Interactive GUI using Tkinter
✔ Automatic abnormal vitals detection
✔ Health alert system

# 📊 Patient Parameters Used

The system analyzes the following patient vitals:

* Heart Rate (bpm)
* SpO2 Level (%)
* Systolic Blood Pressure (mmHg)
* Diastolic Blood Pressure (mmHg)
* Body Temperature (°C)
* Fall Detection Status

These parameters are processed by the machine learning model to predict the patient's health condition.

---

# ⚙️ How It Works

1️⃣ User enters patient vital data in the GUI.

2️⃣ The system processes the data and creates alert flags such as:

* Heart Rate Alert
* SpO2 Alert
* Blood Pressure Alert
* Temperature Alert

3️⃣ The trained **Random Forest model** predicts the patient's health condition.

4️⃣ The predicted condition is displayed in the interface.

5️⃣ **Google Gemini AI** analyzes the vitals and prediction to provide safe health insights and suggestions.
